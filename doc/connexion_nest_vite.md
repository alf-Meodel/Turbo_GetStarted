# Introduction

- a la racine nous pouvons démarrer le server Nest 

```
npm run dev:api
```

- ou Vite react : 

```
npm run dev:web
```

- et si nous voulons lancer en parallèle le back et le frotn nosu allosn faire : 

```
npm run dev
```
